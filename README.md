# Irrigation Crew Scheduler

## Problem & Business Impact
My friend’s irrigation crew was wasting 1-2 hours every morning manually building daily schedules. I built a live Excel system that auto-generates the entire day’s route, crew assignments, production tracking, and inventory — all with one-click Google Maps / Apple Maps links to every stop. It now saves him roughly 5-10 hours per week.

## What I Built
- Master client list
- Daily schedule that auto-populates every crew member’s stops, times, and map links
- Real-time production logging and inventory management
- Fully functional, used in production every single day

## Tech & Approach
- Advanced Excel: formulas, data validation, hyperlinks, queries, conditional formatting
- Vibe-coded the entire logic and structure with Grok while I owned the architecture and edge cases
- Designed for non-technical users — my buddy runs the whole business from this one file.
